# pull

在不同操作系统中，用户目录的环境变量如下：

### Windows
- 环境变量：`%USERPROFILE%`
- 示例路径：`C:\Users\<用户名>`

### macOS 和 Linux
- 环境变量：`$HOME`
- 示例路径：
  - macOS: `/Users/<用户名>`
  - Linux: `/home/<用户名>`

```sh
git clone git@github.com:Nahida-aa/dotfiles.git
```
