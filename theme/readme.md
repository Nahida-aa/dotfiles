## 任务栏

- TranslucentTB

高级颜色选择器 (color picker)，支持更改透明度以及实时预览以更改任务栏的颜色

五种任务栏状态：正常，不透明，透明，模糊，以及亚克力

六种动态状态：可见窗口，最大化窗口，打开开始菜单，打开搜索，打开任务视图，以及省电模式

在 Windows 10 上，可根据动态状态显示或隐藏显示桌面（Aero Peek）按钮

在 Windows 11 上，可根据动态状态显示或隐藏任务栏细线
