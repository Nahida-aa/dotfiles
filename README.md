# migrate

## visual studio code

VS Code的设置同步功能会从你的Microsoft账户或者GitHub账户同步数据。这个功能会同步以下数据：

- 设置（Settings）
- 键盘快捷方式（Keyboard Shortcuts）
- 用户片段（User Snippets）
- 扩展（Extensions）
- 工作区（Workspaces）

当你在一个新的设备上安装VS Code并登录你的账户后，所有的这些设置和数据都会被自动同步到新设备上。
